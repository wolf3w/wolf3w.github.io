OpenMP

Если обрабатывать массив объектов и вызвать в параллельной секции метод у объекта, то возможен segmentation fault если внутри метода есть указатель на метод класса и даже если он 100% инициализирован, столкнулся на работе: #pragma omp parallel for ... arr[i].getPressure(params); В getPressure(): double (*BHPC::solver) (); solver = BHPC::RungeKutta(); ... return (this->*solver)(); // SIGSEGV Segmentation fault
